# MIDI parser

This python script is meant for easily converting MIDI files to Bremen Coq source code.  

## Függőségek
 - Mido 1.2.10  
 - Music21

## Usage

Printing Coq source code:  
```shell
$ python parser.py example.mid
```

Saving Coq source code:  
```shell
$ python parser.py example.mid example.v
```
